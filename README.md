# AllToAll 功能指南

AllToAll 是一款极致简约的全能文件转换工具，旨在为您提供最纯粹、高效的格式处理体验。

## 核心功能

- **多文件批处理**: 支持同时拖入多个文件进行一键批量转换。
- **直观拖拽界面**: 极简设计，支持直接拖拽文件上传。
- **智能格式推荐**: 根据输入文件自动列出最合适的转换目标。
### 多媒体处理能力
- **视频提音**: 支持从常见的视频格式（.mp4, .mkv, .mov）中快速提取高品质音频（.mp3, .wav, .aac）。
- **办公处理**: 支持 Word (.docx) 转 PDF，以及 Excel (.xlsx) 与 CSV 的互转。
- **极速转换**: 利用 FFmpeg、Whisper、Pandas 等引擎，瞬间完成处理。
- **实时格式搜索**: 支持对目标转换格式进行快速搜索和过滤。
- **原名保留选项**: 可自由选择转换后的文件是否保留原始名称。
- **中英文一键切换**: 适配全球用户的流畅使用体验。

## 支持的转换格式

### 文档类 (支持双向转换)
- **Markdown (.md)** ↔ Word (.docx), HTML
- **Word (.docx)** ➔ PDF, Markdown (.md)
- **表格类**: Excel (.xlsx) ↔ CSV

### 媒体类
- **视频提音**: MP4 / MKV / MOV ➔ MP3, WAV, AAC
- **语音转文字**: MP3 / MP4 / WAV / MKV ➔ Markdown (.md), Text (.txt)

## 操作步骤
0. **启动**: 运行start.bat文件,稍等会弹出终端以运行界面。
1. **上传**: 拖入想要转换的文件。
2. **选择**: 搜索并点击目标格式。
3. **转换**: 点击按钮，等待下载。
